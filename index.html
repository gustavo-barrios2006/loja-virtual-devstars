Para navegar em nossa loja:

  1. Escolha o produto desejado
  2. Clique no botão adicionar ao carrinho para adicioná-lo no carrinho de compras
  3. Insira a quantidade desejada
  4. Vá para o carrinho clicando no link correspondente no cabeçalho dessa página
  5. Faça as operações desejadas, como verificar o carrinho, remover itens se desejado, e quando estiver pronto basta clicar em finalizar compra no carrinho para comprar os produtos.
